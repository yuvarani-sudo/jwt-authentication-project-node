# 🔐 JWT Authentication System (Node.js)

A secure authentication system built using Node.js and JSON Web Tokens (JWT). This project includes user registration, login, and protected routes with token-based authentication.

---

## 🚀 Features

- User Registration
- User Login Authentication
- JWT Token Generation
- Protected Routes using Middleware
- Secure Password Handling
- REST API Implementation

---

## 🛠️ Tech Stack

- Node.js
- Express.js
- JWT (JSON Web Token)
- MongoDB (if used)
- bcrypt (for password hashing)
